## 1. キーポイントのおさらい

### 1.1 変数

* 変数を使うには「宣言(せんげん)」して「初期化」する必要がある<br>
  変数の宣言は次のように書く<br>
  &emsp;<code><span class="hljs-keyword">変数の型</span> 変数の名前 = 初期値;</code>
* 変数にデータを書き込む(代入する)には、`=`(イコール)記号を使って次のように書く<br>
  &emsp;<code>変数 = 数値、文章、別の変数など;</code>
* 変数は計算式に組み込める。<br>
  &emsp;<code>変数A = 1 + 変数B * 3;</code>
* `Console.WriteLine`メソッドを使うと、変数の内容をコンソール画面に出力できる<br>
  &emsp;<code><span class="hljs-keyword">int</span> number = 57;</code><br>
  &emsp;<code>Console.WriteLine(number);</code><br>
  このプログラムを実行すると、コンソール画面に`57`と表示される
* `Console.ReadLine`メソッドを使うと、入力された文字や数字を、変数に代入できる<br>
  &emsp;<code><span class="hljs-keyword">string</span> data;</code><br>
  &emsp;<code>Console.ReadLine(data);</code><br>
  &emsp;<code>Console.WriteLine(data);</code><br>
  このプログラムを実行して何かを入力し、Enterキーを押すと、コンソール画面に入力した文字が表示される

### 1.2 型

* 「型」や「クラス」はデータの種類のことで、 整数用の`int`(イント)、実数用の`float`(フロート)、<br>
  文章用の`string`(ストリング)などがある
* 型ごとに「できること(機能)」が決まっている<br>
  例えば、`int`型の変数に代入できるのは「整数」だけで、実数や文字列は代入できない<br>
  `float`型の変数には「実数」を代入できる(実数には整数も含まれる)が、文字列は代入できない<br>
  `string`型の変数には「文字列」を代入できるが、整数や実数は代入できない
* `float`型に代入する実数には、末尾に`f`(小文字のエフ)を付ける必要がある<br>
  &emsp;<code><span class="hljs-keyword">float</span> a = 3.14f;</code>
* `string`型の加算は「2つの文字列を連結した文字列」になる

### 1.3 if文

* if文を使うと「ある条件が正しい時だけ処理をする」というプログラムが書ける
* if文は次のように書く<br>
  <code><span class="hljs-keyword">if</span> (条件式) {<br>
  &emsp;&emsp;処理<br>
  &emsp;}</code>
* else句を使うと「条件式が正しくなかった時」の処理を書ける
* else句は次のように書く<br>
  <code><span class="hljs-keyword">if</span> (条件式) {<br>
  &emsp;&emsp;処理A<br>
  &emsp;} <span class="hljs-keyword">else</span> {<br>
  &emsp;&emsp;処理B<br>
  &emsp;}</code>
* 条件式が真のときは`true`、偽のときは`false`になる
* if文の中にif文を書いて、多重if文を作ることができる
* 論理演算子(`&&`や`||`)を使うと、複雑な条件をあらわすことができる

### 1.4 実行順序とスコープ

* プログラムは上から下に向かって、順番に実行される
* if文やelse句を使うと、プログラムの流れを分岐し、その中からひとつの流れを選べるようになる
* if文では、より厳しい条件を先にチェックする
* `{`と`}`で囲われた範囲を「ブロック」という
* 変数が使える範囲を「スコープ」という
* スコープは、「変数の宣言」で始まり、「同じレベルのブロックの`}`」で終わる

### 1.5 配列

* 配列は「同じ型のデータを複数まとめて扱う」機能
* 配列の宣言には次の２種類の書きかたがある<br>
  <code><span class="hljs-keyword">型</span>[] 配列名 = { データ0, データ1, ... };</code><br>
  <code><span class="hljs-keyword">型</span>[] 配列名 = <span class="hljs-keyword">new</span> 型[データ数];</code>
* 配列の`a`番目の変数を読み書きするには次のように書く<br>
  <code>配列名[a];</code><br>
  このとき、`[`と`]`の中に書く数値のことを「添字(そえじ)」という
* 添字は`0`から数える<br>
  例えば長さ`5`の配列の場合、有効な添字は`0`,`1`,`2`,`3`,`4`で、`5`は使えない
* 配列名に続けて`.Length`(ドット・レングス)を書くと、配列の要素数を調べられる

### 1.6 for文

* for文の「真の能力」を使うには、`()`の中に次のようなプログラムを書く<br>
  &emsp;<code><span class="hljs-keyword">for</span> (<span class="hljs-keyword">int</span> i = 0; i < n; i++)</code><br>
  &emsp;<code>{</code><br>
  &emsp;<code>&emsp;繰り返したいプログラム</code><br>
  &emsp;<code>}</code><br>
  ()内の変数`i`は「何回目の繰り返しか」を記録するもので、「カウンタ変数」や「ループ変数」と呼ばれる<br>
  `n`は「繰り返し回数」をあらわし、整数または変数で指定できる
* for文のループ変数を配列の添字にすると、配列全体、または部分に対して同じ処理を実行できる<br>
  &emsp;<code><span class="hljs-keyword">string</span>[] s = { "春", "夏", "秋", "冬" };</code><br>
  &emsp;<code><span class="hljs-keyword">for</span> (<span class="hljs-keyword">int</span> i = 0; i < s.Length; i++)</code><br>
  &emsp;<code>{</code><br>
  &emsp;<code>&emsp;Console.WriteLine(s[i]);</code><br>
  &emsp;<code>}</code><br>
  このプログラムの出力は、<br>
  &emsp;<code>春</code><br>
  &emsp;<code>夏</code><br>
  &emsp;<code>秋</code><br>
  &emsp;<code>冬</code><br>
  となる
